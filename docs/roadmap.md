# Roadmap

## Phase 1: Documentation + Manual Review System

- Define product vision and architecture.
- Create review skills and runbooks.
- Create sample project config, env inventory, and reports.
- Establish Vault Control Room and Vault Audit foundations.

## Phase 2: CLI Review Runner

- Add local scanner.
- Add change classifier.
- Add deterministic review routing.
- Generate local reports and confidence history.

## Phase 3: API Service

- Add project registry API.
- Add scan submission API.
- Add findings and report APIs.
- Support API-first integration with external projects.

## Phase 4: Control Room Dashboard

- Build Dev Control Room.
- Build Architecture Control Room.
- Build Vault Control Room.
- Show trends, blockers, evidence, and recommended solution prompts.

## Phase 5: Multi-Project Productized Platform

- Support organizations and multiple projects.
- Add hosted evidence storage.
- Add role-based access.
- Add aggregate learning from privacy-safe operational metadata.
